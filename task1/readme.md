# 1.什么是Pytorch，为什么选择Pytroch？
- 作为NumPy的替代品，可以利用GPU的性能进行计算
- 作为一个高灵活性、速度快的深度学习平台
- Pytorch相比Tensorflow而言，它的设计初衷是简单易用用，所以它是基于动态图进行实现的，从而方便调试。当然，Tensorflow在1.5版的时候就引入了Eager Execution机制实现了动态图，但它还是默认使用静态图。
